*General Movement*
- `Super + ←`  Tile window to left half
- `Super + →`  Tile window to right half
- `Super + ↑`  Maximize window  ## <- broken?
- `Super + ↓`  Restore / unmaximize window  ## <- broken?

*Move Window Between Monitors*
- `Shift + Super + ←`  Move window to monitor on the left
- `Shift + Super + →`  Move window to monitor on the right
- `Shift + Super + ↑`  Move window to monitor above (if applicable)
- `Shift + Super + ↓`  Move window to monitor below (if applicable)

*Workspace Navigation*
- `Super + Page Up`  Switch to workspace above
- `Super + Page Down`  Switch to workspace below

*Move Window Between Workspaces*
- `Shift + Super + Page Up`  Move window to workspace above
- `Shift + Super + Page Down`  Move window to workspace below

*Alternative Workspace Shortcuts (often enabled)*
- `Ctrl + Alt + ↑`  Switch workspace up
- `Ctrl + Alt + ↓`  Switch workspace down
- `Ctrl + Alt + Shift + ↑`  Move window up a workspace
- `Ctrl + Alt + Shift + ↓`  Move window down a workspace

*Keyboard Move Mode*
- `Alt + F7`  Move window with keyboard (use arrows, Enter to confirm)
- `Alt + F8`  Resize window with keyboard
