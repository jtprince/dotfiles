^=control + ⇧=shift = kitty-mod

Kitty uses "window" for what feels like a pane.

Create window / pane
kitty-mod \   new window
kitty-mod N   new OS window

Navigate windows / panes
kitty-mod ]   next window
kitty-mod [   previous window
kitty-mod 1   first window
kitty-mod 2   second window
...
kitty-mod 9   ninth window
kitty-mod 0   tenth window

Move windows / panes around
kitty-mod M   move window to top
kitty-mod F   move window forward
kitty-mod B   move window backward

Close window / pane
kitty-mod W

Resize window / pane
kitty-mod R   start interactive resize

Layout
kitty-mod L   next layout

Font size
kitty-mod =   increase
kitty-mod -   decrease
kitty-mod ⌫   reset

Tabs
kitty-mod T   new tab
kitty-mod Q   close tab
kitty-mod →   next tab
kitty-mod ←   previous tab
kitty-mod .   move tab forward
kitty-mod ,   move tab backward

kitty-mod + alt + T  set tab title
